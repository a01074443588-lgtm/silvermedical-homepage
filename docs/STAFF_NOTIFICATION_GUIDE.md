# 직원 알림 사용 안내

## 내 알림 화면 열기

1. Chrome 또는 Edge에서 `https://staff.silvermedical.kr/staff/`에 로그인합니다.
2. `직원 알림 관리`에서 본인 이름을 확인합니다.
3. `기기·카카오 연결 관리`를 누릅니다.

## PC Chrome·Edge

1. `이 기기에서 알림 받기`를 누릅니다.
2. 브라우저가 알림 허용 여부를 물으면 `허용`을 선택합니다.
3. 등록 기기 수가 1대 이상인지 확인합니다.
4. `시험 알림 보내기`를 누르고 운영체제 알림이 도착하는지 확인합니다.

알림이 오지 않으면 주소창 왼쪽의 사이트 설정에서 알림이 차단되어 있지 않은지 확인합니다. Windows의 `설정 > 시스템 > 알림`에서도 Chrome 또는 Edge 알림을 켭니다.

## 안드로이드 Chrome

1. 휴대전화 Chrome에서 직원 관리 주소에 로그인합니다.
2. `내 알림 설정`에서 `이 기기에서 알림 받기`를 누르고 허용합니다.
3. 시험 알림을 보냅니다.
4. 휴대전화의 `설정 > 알림 > 앱 알림 > Chrome`도 허용 상태인지 확인합니다.

같은 직원이 PC와 휴대전화에 각각 등록하면 두 기기에서 모두 받을 수 있습니다.

## 아이폰

아이폰은 iOS 16.4 이상에서 홈 화면에 설치한 웹앱만 푸시 알림을 받을 수 있습니다.

1. Safari에서 직원 관리 주소를 엽니다.
2. 하단 공유 버튼을 누르고 `홈 화면에 추가`를 선택합니다.
3. 홈 화면에 생긴 `실버메디컬 알림` 아이콘으로 다시 엽니다.
4. 로그인 후 `이 기기에서 알림 받기`를 누르고 허용합니다.
5. 시험 알림을 보냅니다.

Safari 탭에서 바로 연 화면에는 아이폰 푸시 등록이 표시되지 않을 수 있습니다.

## 상담 처리

- 알림을 누르면 해당 상담 상세 화면으로 이동합니다.
- 로그아웃 상태면 로그인 후 원래 상담 화면으로 돌아옵니다.
- 상세 화면을 처음 열면 최초 열람자와 시각이 기록되고 재알림이 중단됩니다.
- `접수 확인`, `연락 완료`, `상담 종결` 버튼으로 주요 처리 시각을 기록합니다.
- 담당자, 직원 메모, 진행 상태를 저장하면 처리 이력에 남습니다.

## 직원 추가

1. 최고관리자가 `인증 및 권한 > 사용자`에서 직원 계정을 만들고 `스태프 권한`을 켭니다.
2. 상담 열람·변경과 본인 알림 설정에 필요한 권한만 부여합니다.
3. `직원 알림 관리`에서 직원 이름, 직책, 담당 급여종별, 수신 상담 유형을 설정합니다.
4. 직원 본인이 로그인해 사용하는 기기와 카카오 계정을 연결합니다.
5. 시험 알림으로 실제 수신을 확인합니다.

퇴사자는 먼저 로그인 계정을 비활성화합니다. 그러면 알림 기기와 카카오 연결도 자동 중단됩니다.
